Trong 13 tuần thực tập cùng chương trình AWS First Cloud Journey (08/09 – 04/12/2025), tôi tập trung vào frontend của Nền tảng Hạ tầng Bảo mật IoT và tích hợp với các dịch vụ AWS (DynamoDB, S3, SES, Amplify, Route 53, Cognito, API Gateway, Lambda). Tôi xây dựng luồng giao diện cho xác thực an toàn, truy cập dữ liệu, xử lý file và gửi thông báo, đồng thời bám sát kiến trúc và tiêu chuẩn bảo mật của chương trình. Việc này giúp tôi nâng cao kỹ năng frontend, tích hợp dịch vụ đám mây, phối hợp IaC/DevOps và truyền thông kỹ thuật. Tôi luôn cố gắng giao sản phẩm chất lượng, tuân thủ hướng dẫn FCJ, tham gia cộng đồng AWS và hợp tác hiệu quả với mentor và đội ngũ.

### Tự đánh giá chi tiết

- **Kiến thức và kỹ năng chuyên môn:** Phát triển và tinh chỉnh frontend với React/TypeScript và AWS Amplify; tích hợp Cognito cho auth, API Gateway/Lambda cho backend, DynamoDB cho dữ liệu, S3 cho file, SES cho thông báo, Route 53 cho domain; giữ chất lượng code qua review và lint.
- **Khả năng học hỏi:** Tiếp thu nhanh Amplify hosting, luồng auth Cognito, pattern truy cập DynamoDB, cấu hình SES email, tích hợp API Gateway/Lambda; kiểm chứng qua tài liệu, lab và bài học sự kiện.
- **Chủ động:** Đề xuất và thực hiện cải tiến UX và tự động hóa (auth gọn hơn, xử lý file an toàn, email cảnh báo); tham dự thêm sự kiện AWS để mang best practice về cho đội.
- **Tinh thần trách nhiệm:** Hoàn thành deliverable hàng tuần, ghi worklog chi tiết, đảm bảo vượt qua kiểm thử và quét bảo mật trước khi merge.
- **Kỷ luật:** Tuân thủ lịch FCJ và tham dự đủ 6 sự kiện; một vài đầu việc cần điều chỉnh nhẹ tiến độ nhưng luôn báo sớm.
- **Tính cầu tiến:** Chủ động xin phản hồi mentor về kiến trúc frontend và cách dùng AWS; lặp lại pattern cho auth, lưu trữ, triển khai; áp dụng bài học về tối thiểu đặc quyền và quan sát hệ thống.
- **Giao tiếp:** Viết ghi chú kiến trúc, worklog ngắn gọn, dịch 3 blog kỹ thuật, trình bày cập nhật nền tảng; hướng tới súc tích hơn khi thuyết trình.
- **Hợp tác nhóm:** Phối hợp với đội 5 người, pair-debug các lỗi tích hợp, chia sẻ bài học sự kiện với thực tập sinh khác và cộng đồng AWS.
- **Ứng xử chuyên nghiệp:** Giữ thái độ tôn trọng, xây dựng; tuân thủ chuẩn FCJ; hỗ trợ đồng đội qua review và chia sẻ kiến thức.
- **Tư duy giải quyết vấn đề:** Gỡ lỗi edge case auth Cognito, xử lý/hiệu năng DynamoDB và S3, tinh chỉnh SES/Route 53 để gửi ổn định, tối ưu luồng API Gateway/Lambda.
- **Đóng góp vào dự án/đội:** Hoàn thiện frontend nền tảng bảo mật IoT với tích hợp AWS, dịch 3 blog kỹ thuật, chia sẻ insight từ sự kiện để nâng chất lượng thực hành của đội.
- **Tổng thể:** Cân bằng giữa giao sản phẩm và đóng góp cộng đồng; củng cố kỹ năng frontend tích hợp cloud và nâng cao hợp tác nhóm.
