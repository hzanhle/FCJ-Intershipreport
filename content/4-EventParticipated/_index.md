## Event 1: FCJ 2025 Kickoff Event

**Date:** September 8, 2025 — Full Day  
**Location:** Ho Chi Minh City  
**Role:** Participating Intern

The FCJ 2025 Kickoff Event officially inaugurated the First Cloud Journey internship program. The event gathered interns, mentors, program coordinators, and AWS community leaders for a full-day orientation and networking program. It provided complete clarity on expectations, deliverables, assessment methods, and collaboration channels for the 2025 cohort.

### Key Takeaways
- Understood program structure, milestones, and evaluation framework
- Participated in a project ideation workshop and validated initial IoT Security Infrastructure proposal
- Presented a 5-minute project pitch and received feedback from mentors
- Connected with 25+ interns and 5 mentors specializing in AWS security and IoT
- Developed a roadmap for program completion and project delivery

This kickoff laid the groundwork for my internship by aligning expectations, strengthening confidence, and establishing early mentor-peer connections.
